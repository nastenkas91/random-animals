# Проект random-animals

Антистресс пэт-проект со случайными картинками и гифками котиков, собачек и лисичек. 

## технологии:
* React
* Api

## Запуск проекта
* npm i — установить зависимости;
* npm run dev — запустить проект на локальном сервере;
